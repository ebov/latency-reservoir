# Local clock models

Previous analyses have explored the rate heterogeneity implied by rooting the ebola tree
with the 1970s outbreaks as a cluster.
We have seen that such a rooting is not supported by the data, and proposed a new
root position that is consistent with the between and within cluster evolutionary trends
seen in a subset of the data.

Here, we will extend our previous analysis to determine the extent to which this rooting is robust
to analysis which include all sequenced ebola outbreaks. 
We will apply _a priori_ local clock models as before and assess the support of each model
using PAML and BEAST analyses. 

Previous work has applied similar BEAST models but required the use of restrictive topological 
and/or rate priors to enforce the 1970s root.
